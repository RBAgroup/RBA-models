# *Arabidopsis thaliana* Col-0 (generic cell)  RBA model

The underlying RBA model of *Arabidopsis thaliana* was generated using the RBApy package (...) 
from the previous metabolic reconstruction (https://doi.org/10.1016/j.ymben.2024.03.009).

Please note: due to large file size of the proteome annotation file the file data/proteome_annotation.tsv has been compressed into .zip format.
